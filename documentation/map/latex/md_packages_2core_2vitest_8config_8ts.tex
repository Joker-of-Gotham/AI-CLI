{\ttfamily vitest.\+config.\+ts} 是 core 包的 Vitest 测试配置文件。\+Vitest 是一个基于 Vite 的单元测试框架，这个配置文件定义了测试运行的参数和行为。\hypertarget{md_packages_2core_2vitest_8config_8ts_autotoc_md1062}{}\doxysection{\texorpdfstring{配置详情}{配置详情}}\label{md_packages_2core_2vitest_8config_8ts_autotoc_md1062}

\begin{DoxyItemize}
\item {\ttfamily reporters}\+: 配置测试报告器为默认报告器和 JUnit 报告器
\item {\ttfamily silent}\+: 静默模式运行测试
\item {\ttfamily output\+File}\+: JUnit 报告输出到 {\ttfamily junit.\+xml} 文件
\end{DoxyItemize}\hypertarget{md_packages_2core_2vitest_8config_8ts_autotoc_md1064}{}\doxysubsection{\texorpdfstring{测试设置}{测试设置}}\label{md_packages_2core_2vitest_8config_8ts_autotoc_md1064}

\begin{DoxyItemize}
\item {\ttfamily enabled}\+: 启用代码覆盖率
\item {\ttfamily provider}\+: 使用 V8 作为覆盖率提供者

\item {\ttfamily cobertura}\+: Cobertura XML 格式的报告

该配置文件会被 Vitest 自动加载，用于配置测试运行环境。当运行 {\ttfamily npm test} 或 {\ttfamily vitest} 命令时，会使用此配置来执行测试和生成报告。
