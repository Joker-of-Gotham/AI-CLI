 Calculates diff statistics between three versions of a file\+:
\begin{DoxyEnumerate}
\item Between original ({\ttfamily old\+Str}) and AI-\/proposed ({\ttfamily ai\+Str}) versions
\item Between AI-\/proposed ({\ttfamily ai\+Str}) and user-\/modified ({\ttfamily user\+Str}) versions
\end{DoxyEnumerate}

Returns a {\ttfamily Diff\+Stat} object containing\+:

\item {\ttfamily model\+\_\+removed\+\_\+lines}\+: Lines removed in AI proposal

\item {\ttfamily model\+\_\+removed\+\_\+chars}\+: Characters removed in AI proposal
\item {\ttfamily user\+\_\+added\+\_\+lines}\+: Lines added by user

\item {\ttfamily user\+\_\+added\+\_\+chars}\+: Characters added by user
\item {\ttfamily user\+\_\+removed\+\_\+chars}\+: Characters removed by user
\end{DoxyItemize}
