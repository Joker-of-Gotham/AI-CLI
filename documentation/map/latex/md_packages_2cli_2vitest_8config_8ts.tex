{\ttfamily vitest.\+config.\+ts} 是 cli 包的 Vitest 测试配置文件。\+Vitest 是一个基于 Vite 的单元测试框架，这个配置文件定义了测试运行的参数和行为。\hypertarget{md_packages_2cli_2vitest_8config_8ts_autotoc_md479}{}\doxysection{\texorpdfstring{配置详情}{配置详情}}\label{md_packages_2cli_2vitest_8config_8ts_autotoc_md479}
