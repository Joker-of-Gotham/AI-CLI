\begin{DoxyItemize}
\item 启动 CLI 应用程序的主入口点
\item 捕获并处理致命错误
\item 根据错误类型进行不同的处理：
\begin{DoxyItemize}
\item 如果是 {\ttfamily Fatal\+Error} 类型，显示错误信息并以相应退出码退出
\item 如果是其他错误，显示堆栈跟踪并以 1 退出
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_packages_2cli_2index_8ts_autotoc_md174}{}\doxysection{\texorpdfstring{错误处理}{错误处理}}\label{md_packages_2cli_2index_8ts_autotoc_md174}

\begin{DoxyItemize}
\item 使用 {\ttfamily Fatal\+Error} 类型来处理已知的致命错误
\item 对于未知错误，显示完整的堆栈跟踪信息
\item 支持通过 {\ttfamily NO\+\_\+\+COLOR} 环境变量控制错误信息的颜色显示 
\end{DoxyItemize}
