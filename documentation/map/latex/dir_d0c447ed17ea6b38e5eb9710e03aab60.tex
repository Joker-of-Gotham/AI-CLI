\hypertarget{README.md_autotoc_md1070}{}\doxysubsection{\texorpdfstring{Gemini CLI Companion}{Gemini CLI Companion}}\label{README.md_autotoc_md1070}
The Gemini CLI Companion extension pairs with \href{https://github.com/google-gemini/gemini-cli}{\texttt{Gemini CLI}}. This extension is compatible with both VS Code and VS Code forks.\hypertarget{README.md_autotoc_md1071}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{README.md_autotoc_md1071}

\begin{DoxyItemize}
\item Open Editor File Context\+: Gemini CLI gains awareness of the files you have open in your editor, providing it with a richer understanding of your project\textquotesingle{}s structure and content.
\item Selection Context\+: Gemini CLI can easily access your cursor\textquotesingle{}s position and selected text within the editor, giving it valuable context directly from your current work.
\item Native Diffing\+: Seamlessly view, modify, and accept code changes suggested by Gemini CLI directly within the editor.
\item Launch Gemini CLI\+: Quickly start a new Gemini CLI session from the Command Palette (Cmd+\+Shift+P or Ctrl+\+Shift+P) by running the "{}\+Gemini CLI\+: Run"{} command.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md1072}{}\doxysubsection{\texorpdfstring{Requirements}{Requirements}}\label{README.md_autotoc_md1072}
