\hypertarget{md_packages_2vscode-ide-companion_2development_autotoc_md1068}{}\doxysection{\texorpdfstring{Running the Extension}{Running the Extension}}\label{md_packages_2vscode-ide-companion_2development_autotoc_md1068}
To run the extension locally for development\+:


\begin{DoxyEnumerate}
\item From the root of the repository, install dependencies\+: {\ttfamily bash npm install }
\item Open this directory ({\ttfamily packages/vscode-\/ide-\/companion}) in VS Code.
\item Compile the extension\+: {\ttfamily bash npm run compile }
\item Press {\ttfamily F5} (fn+f5 on mac) to open a new Extension Development Host window with the extension running.
\end{DoxyEnumerate}

To watch for changes and have the extension rebuild automatically, run\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ run\ watch}

\end{DoxyCode}
