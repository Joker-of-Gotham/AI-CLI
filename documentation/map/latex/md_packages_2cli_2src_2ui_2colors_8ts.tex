\item {\bfseries{属性}}\+:

\item {\ttfamily type}\+: 颜色主题类型

\item {\ttfamily Background}\+: 背景色
\item {\ttfamily Light\+Blue}\+: 浅蓝色
\item {\ttfamily Accent\+Blue}\+: 强调蓝色

\item {\ttfamily Accent\+Yellow}\+: 强调黄色

\item {\ttfamily Diff\+Removed}\+: 差异删除颜色
\item {\ttfamily Comment}\+: 注释颜色
\item {\ttfamily Gray}\+: 灰色
\item {\ttfamily Gradient\+Colors}\+: 渐变颜色数组
\end{DoxyItemize}
